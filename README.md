# Виртуальная галерея природы

Прогулка от первого лица по залам с фотографиями природы на компьютере.

---

## Как скачать и запустить (для новичков)

Нужна обычная **Windows 10 или 11**. Больше ничего устанавливать не нужно.

### 1. Скачайте

1. Откройте: [https://github.com/nsasa0474-lgtm/virtual-nature-gallery](https://github.com/nsasa0474-lgtm/virtual-nature-gallery)
2. Нажмите зелёную кнопку **Code** → **Download ZIP**
3. Распакуйте архив куда угодно (например на Рабочий стол)

### 2. Запустите

1. Откройте папку с проектом
2. Дважды щёлкните **`Запустить_галерею.bat`**  
   (или **`Start-Gallery.bat`** — то же самое)
3. Подождите пару секунд — сам откроется браузер

### 3. Как выйти

Просто **закройте чёрное окно** — галерея остановится.

---

## Управление в галерее

| Клавиши | Действие |
| --- | --- |
| **W A S D** или стрелки | Ходьба |
| Мышь | Осмотр |
| **Esc** | Отпустить курсор |
| **E** у южной двери | Пароль секретной комнаты |
| **L** | Запереть секретную комнату |

На стартовом экране дождитесь, пока загрузятся фото, затем нажмите **«Войти в галерею»**.

---

## Как поменять все фото на свои

1. Откройте папку **`new_foto`**
2. Положите туда свои картинки (`.jpg`, `.png`, `.webp`, `.gif`)
3. Дважды щёлкните **`smena_foto.bat`**
4. Дождитесь «Готово»
5. Снова запустите **`Запустить_галерею.bat`**

Если папка пустая или фото те же — ничего не меняется.

---

## Если что-то не работает

- Браузер не открылся — посмотрите адрес в чёрном окне (обычно `http://127.0.0.1:8765/`) и откройте его вручную.
- Антивирус ругается на `.bat` — разрешите запуск для этой папки.
- Пишет, что нет `dist\index.html` — скачайте ZIP заново с GitHub (не удаляйте папку `dist` из архива).
- На работе/учёбе режут localhost — попробуйте дома на своём ПК.

## Для разработчиков (необязательно)

Если меняете исходники в `src/`, нужен [Node.js](https://nodejs.org):

```bash
npm install
npm run build
npm start
```

Стек: Vite + Three.js. Секретная комната: `public/secret/` (AES-GCM); исходники секретов только локально в `nopublic/`.

# Виртуальная галерея природы

Прогулка от первого лица по залам с фотографиями природы на компьютере.

---

## Как скачать и запустить (для новичков)

### Шаг 1. Установите Node.js

1. Откройте сайт: [https://nodejs.org](https://nodejs.org)
2. Скачайте версию **LTS** (зелёная кнопка).
3. Установите, как обычную программу (везде «Далее» / Next).
4. **Перезагружать компьютер не обязательно**, но если что-то не найдётся — перезапустите его.

Проверка: откройте «Командную строку» или PowerShell и введите:

```text
node -v
```

Должна появиться версия, например `v22.…`. Если пишет, что команда не найдена — Node.js не установился или нужен перезапуск.

### Шаг 2. Скачайте галерею с GitHub

**Вариант А (проще всего):**

1. Откройте страницу проекта: [https://github.com/nsasa0474-lgtm/virtual-nature-gallery](https://github.com/nsasa0474-lgtm/virtual-nature-gallery)
2. Нажмите зелёную кнопку **Code** → **Download ZIP**
3. Распакуйте архив в любую папку, например на Рабочий стол

**Вариант Б (если умеете git):**

```text
git clone https://github.com/nsasa0474-lgtm/virtual-nature-gallery.git
```

### Шаг 3. Запустите

1. Откройте папку с проектом
2. Дважды щёлкните файл **`Start-Gallery.bat`**  
   (или **`Запустить_галерею.bat`** — то же самое)
3. Подождите: при **первом** запуске программа сама поставит нужные файлы и соберёт галерею (может занять пару минут)
4. Сам откроется браузер с адресом вроде `http://127.0.0.1:8765/`

### Шаг 4. Как выйти

Закройте чёрное окно консоли — сервер остановится.

## Управление в галерее

| Клавиши | Действие |
| --- | --- |
| **W A S D** или стрелки | Ходьба |
| Мышь | Осмотр |
| **Esc** | Отпустить курсор |
| **E** у южной двери | Пароль секретной комнаты |
| **L** | Запереть секретную комнату |

На стартовом экране дождитесь надписи, что фото загружены, затем нажмите **«Войти в галерею»**.

---

## Как поменять все фото на свои

1. Откройте папку **`new_foto`**
2. Положите туда свои картинки (`.jpg`, `.png`, `.webp`, `.gif` — сколько угодно)
3. Вернитесь в основную папку проекта и дважды щёлкните **`smena_foto.bat`**
4. Дождитесь сообщения «Готово»
5. Снова запустите **`Start-Gallery.bat`**

Если в `new_foto` пусто или фото те же самые — галерея не меняется.

---

## Если что-то не работает

- **«Node.js не найден»** — установите Node.js (шаг 1) и закройте/откройте окно заново.
- **Браузер не открылся** — вручную откройте адрес из чёрного окна, обычно `http://127.0.0.1:8765/`
- **Антивирус ругается** — разрешите запуск `.bat` / Node для этой папки.
- **Хотите вернуть стандартные 100 фото природы** — в папке проекта откройте терминал и выполните:

```text
npm run download-photos
npm start
```

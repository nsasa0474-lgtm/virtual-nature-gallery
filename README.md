# virtual-nature-gallery

Когда вечер застаёт без срочных дел, остаётся лишь один честный жест — создать пространство для созерцания. Так появилась эта виртуальная галерея природы: стены из ничего, свет из пикселей, и сто пейзажей, среди которых можно просто бродить.

*Дело было вечером. Делать было нечего. Пришлось искать смысл в тишине залов.*

## Запуск

Нужен [Node.js](https://nodejs.org).

```bash
npm install
npm run build
```

Затем дважды кликните **`Запустить_галерею.bat`** (Windows) или:

```bash
npm start
```

Закройте окно консоли, чтобы остановить сервер.

## Управление

- **WASD** / стрелки — ходьба  
- Мышь — осмотр  
- **Esc** — отпустить курсор  
- У южной двери: **E** — пароль секретной комнаты, **L** — запереть (ключ и текстуры сбрасываются)

Секретные фото лежат в `public/secret/` только в зашифрованном виде (AES-GCM); исходники — в локальной `nopublic/` (в git не попадает). Ограничение: после успешного ввода пароля экран и запись монитора не защищаются — защищается скачанный клиент/репозиторий без пароля.

Перешифровать: `SECRET_PASSWORD=... npm run encrypt-secret` (пароль в код и git не записывать).

## Стек

Vite + Three.js, локальные фото природы, первый взгляд от лица посетителя музея.

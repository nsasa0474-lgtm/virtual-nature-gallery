# virtual-nature-gallery

Когда вечер застаёт без срочных дел, остаётся лишь один честный жест — создать пространство для созерцания. Так появилась эта виртуальная галерея природы: стены из ничего, свет из пикселей, и сто пейзажей, среди которых можно просто бродить.

*Дело было вечером. Делать было нечего. Пришлось искать смысл в тишине залов.*

---

## Как скачать и запустить (для новичков)

Нужна обычная **Windows 10 или 11**. Ничего устанавливать не нужно (Node.js не нужен).

### 1. Скачайте

1. Откройте: [https://github.com/nsasa0474-lgtm/virtual-nature-gallery](https://github.com/nsasa0474-lgtm/virtual-nature-gallery)
2. Нажмите зелёную кнопку **Code** → **Download ZIP**
3. **Обязательно распакуйте** архив (не открывайте bat изнутри ZIP)

### 2. Запустите

1. Откройте распакованную папку
2. Дважды щёлкните **`Запустить_галерею.bat`** (или **`Start-Gallery.bat`**)
3. Если Windows SmartScreen спросит — **Подробнее** → **Выполнить в любом случае**
4. Подождите пару секунд — откроется браузер

Адрес подберётся сам (свободный порт). Он также пишется в файл `gallery-url.txt`.

### 3. Как выйти

Закройте чёрное окно — галерея остановится.

## Управление

| Клавиши | Действие |
| --- | --- |
| **W A S D** или стрелки | Ходьба |
| Мышь | Осмотр |
| **Esc** | Отпустить курсор |
| **E** у южной двери | Пароль секретной комнаты |
| **L** | Запереть секретную комнату |

Дождитесь загрузки фото, затем нажмите **«Войти в галерею»**.

---

## Свои фото

1. Положите картинки в **`new_foto`** (`.jpg` `.png` `.webp` `.gif`)
2. Запустите **`smena_foto.bat`** (галерею перед этим закройте)
3. Снова **`Запустить_галерею.bat`**

---

## Если что-то не так

Сообщение об ошибке будет в чёрном окне — его обычно достаточно.

Частые случаи:

- Браузер не открылся → откройте адрес из окна или из `gallery-url.txt`
- SmartScreen / антивирус → разрешите запуск для этой папки
- «Нет dist\index.html» → скачайте ZIP заново, не удаляйте папку `dist`
- Порт занят → программа сама ищет свободный; если пишет, что портов нет — перезагрузите ПК
- Запуск из ZIP без распаковки → так нельзя, сначала извлеките папку
